# Docker PHP Project - PHP 7.2

## 1. Extensões do PHP 7.2

De forma muito fácil, é possível instalar/desinstalar extensões do PHP através das 
configurações no arquivo docker.php.

Por exemplo:

```php
php('7.2')
    ->param('name', 'app')
    ->extension('mysql')
    ->extension('gd')
```
